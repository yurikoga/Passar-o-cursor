# 3D Flip Card on Hover using only HTML & CSS | CodingLab

Um projeto simples de **card com efeito flip 3D no hover** usando apenas HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **card interativo com efeito de rotação 3D**, revelando o verso ao passar o mouse.

## 🛠️ Tecnologias

- HTML
- CSS (transform, transition, perspective)

## 🎯 Funcionalidades

- Efeito 3D no hover
- Animação suave
- Design moderno e interativo

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
